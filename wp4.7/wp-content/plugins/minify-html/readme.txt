== Description ==
Allows you compress HTML, this plugin solution help you saving size your page.

== Installation ==

1. Upload the `minify-html` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the Plugins menu in WordPress. After activation, you should see HTML your page compressed in front-end.

**Enjoy!**
