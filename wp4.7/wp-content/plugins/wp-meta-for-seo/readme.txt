=== WordPress Meta For SEO ===
Author: cuongnq@greenglobal.vn
Tags: Meta SEO, Google Authorship, Structure Data Markup, Facebook Admin
Requires at least: 2.9.0
License: Free

Allows you add custom field in Post: Short Description, Meta Description, Meta Title, published_at (integer), updated_at (integer)

== Description ==
Display Meta SEO in head, include 4 group.
1. Normal (Charset, title, description, robots, link rel canonical)
2. Google Authorship (rel publisher and rel author)
3. Schema.org markup for Google (itemprop: name, description, image)
4. Graph data (include meta property as og:title, og:title, og:locale, og:type, og:url, og:image, og:description, og:site_name, fb:admins, og:updated_time, article:published_time, article:modified_time, article:section, article:tag)

== Installation ==

1. Upload the `wp-meta-for-seo` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the Plugins menu in WordPress. After activation, you should see Meta SEO display in head of front-end.
3. Configure the plugin.
4. Make sure you have updated "Your Facebook Admin ID" and "Your Google Plus User ID [Google+]" in Settings -> General.
5. Go to home and F5 refresh page you will see google authorship and fb:admins display in <head>.

**Enjoy!**
